DOM Tree: refers to the html page where all the nodes are objects. There are 3 types of nodes:
  1. Text Nodes
  2. Element Nodes
  3. Comment Nodes

In html page, <html> is the root and <head> and <body> are its children

NOTE: A text node is always a leaf node of the tree
