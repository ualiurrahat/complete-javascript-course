## üéÆ How It Works

1. **Click any "Show modal" button** (1, 2, or 3)
2. **Modal appears** with blurred backdrop
3. **Close options**:
   - Click **√ó** (top-right)
   - Click **anywhere on the overlay**
   - Press **Esc key** on keyboard
4. **Repeat** from any trigger button!

> üí° **Pro Tip**: Try opening from different buttons - same modal, different entry points!
